# Hamburbur
### A Mod Menu for Gorilla Tag

Hamburbur is a simple, clean, and easy-to-use mod menu for Gorilla Tag.  
No clutter. No overcomplication. Just a quality modding experience.

---

## Overview

Hamburbur focuses on simplicity above everything else.  
The goal is to provide a clean and minimal mod menu experience without unnecessary features or visual noise.

---

## Features

- Advanced customization
- AI-powered voice assistant
- Straightforward navigation
- Core quality-of-life features
- No unnecessary bloat

---

## Installation

1. Install BepInEx (5.x recommended).
2. Place `Hamburbur.dll` inside your `BepInEx/plugins` folder.
3. Launch Gorilla Tag.
4. Open the menu in-game using:
   - `Y` (VR)
   - `Tab` or `H` (PC)

---

## Requirements

- Gorilla Tag (Steam VR version (rift is not supported)
- BepInEx 5.x
- Any OS that can support BepInEx and GorillaTag (Windows is needed for Jarvis or Voice Controls)
